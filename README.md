# Skincare Ingredient Recommendation Chatbot

An interactive Python and Streamlit application that helps users choose the right skincare ingredients based on their skin concerns and previous product usage.


## Features

- Select skin concern (acne, dry skin, pigmentation, sensitivity, etc.)
- Shows recommended ingredients
- Shows ingredients to avoid.
- Takes skincare history input.
- Simple and clean web interface.


## Tech Stack

- Python  
- Pandas  
- Streamlit
